1- Create new folder "JTO", fork the project in Github and clone it 
2- Create new python enviroment : "python -m venv env" ( for python 3 and above, virtualenv will be installed) (http://flask.pocoo.org/docs/1.0/installation/)
3- Active the enviroment first then install Flask : "pip install Flask". 
4- Intall Postgresql version 10.9 (https://www.enterprisedb.com/downloads/postgres-postgresql-downloads) 
5- Add the installation path ( bin folder) to the windows Path, so you can run "psql" from the command line (https://realpython.com/flask-by-example-part-2-postgres-sqlalchemy-and-alembic/)
6- To login to the superuser : psql -U postgres and the password you entered in the installation
7- create database JTO using psql command line or from pgadmin 
8- Run these commands : (https://blog.theodo.com/2017/03/developping-a-flask-web-app-with-a-postresql-database-making-all-the-possible-errors/)
    -pip install psycopg2 
    -pip install Flask-SQLAlchemy
    -pip install Flask-Migrate
    -pip install flask_migrate
    -pip install flask_script
9- pip freeze > requirements.txt (to generate the requirements file that shows the version of each package)
10- Run the following to prepare the database : 
    - python manage.py db init (one time only)
    - python manage.py db migrate
    - python manage.py db upgrade
11- At this point, a new table called "users" will be create automatically in the db
